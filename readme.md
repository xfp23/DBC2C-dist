# DBC2C —— 让 CAN 协议开发回归工程本身

DBC2C 是一款面向汽车电子、工业控制及嵌入式开发的 CAN 协议工程工具。

它以 Excel 作为协议设计入口，实现协议文件、DBC 文件与代码之间的自动转换，让工程师摆脱重复、低效、容易出错的手工维护工作，把更多时间投入到真正有价值的开发中。

---

## 为什么选择 DBC2C？

在实际项目中，协议通常以 Excel 的形式在客户、硬件工程师、软件工程师和测试工程师之间流转。

传统流程往往意味着：

- 手工制作 DBC 文件
- 手工维护协议文档
- 手工编写解析代码
- 多份数据重复维护
- 一个字段修改后，需要同步多个文件

任何一步遗漏，都可能导致协议错误、调试困难甚至项目延期。

DBC2C 将这些重复劳动全部自动化。

---

## 核心功能

### Excel → DBC

根据统一的 Excel 协议模板，一键生成标准 DBC 文件。

无需重复录入协议信息，避免人工制作 DBC 带来的错误。

---

### DBC → C 代码（专业版）

自动生成可直接使用的 C 代码，快速完成 CAN 报文收发、信号解析等工作。

与 MicroCAN 运行库配合使用，可进一步减少重复开发工作量。

---

### DBC → Excel

支持将已有 DBC 文件快速导出为 Excel，方便协议整理、版本维护以及客户交付。

---

### 协议浏览器

内置高速协议浏览界面，支持：

- 报文浏览
- 信号浏览
- 实时搜索
- 快速定位

面对大型 DBC 文件依然保持流畅体验。

---

## 为大型项目而设计

采用异步解析与虚拟列表技术，即使面对包含数千个报文、上万个信号的协议文件，也能够保持界面流畅响应。

无需等待，无需卡顿。
